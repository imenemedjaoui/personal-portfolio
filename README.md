# Imène Medjaoui — Personal Portfolio

A modern, dark-themed personal portfolio website showcasing my experience, skills, and projects as a Full Stack Developer.

## Preview

Single-page portfolio featuring smooth scroll animations, interactive card effects, and a fully responsive design.

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Vanilla ES6+
- **GSAP** — ScrollTrigger-powered scroll animations
- **Google Fonts** — Inter & JetBrains Mono

## Features

- Dark mode UI with purple/blue gradient accents
- GSAP scroll-triggered entrance animations
- Cursor glow effect that follows mouse movement
- Magnetic hover effect on buttons
- 3D tilt effect on cards
- Animated stat counters
- Floating parallax gradient orbs
- Fully responsive (mobile hamburger menu)
- Smooth scroll navigation with active link highlighting
